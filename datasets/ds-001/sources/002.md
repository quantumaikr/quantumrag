# Python `collections` Module Documentation

## Overview

The `collections` module implements specialized container datatypes providing alternatives to Python's built-in containers (`dict`, `list`, `set`, `tuple`).

## Container Types

| Type | Purpose |
|------|---------|
| `namedtuple()` | Factory function for tuple subclasses with named fields |
| `deque` | List-like container with fast appends/pops on either end |
| `ChainMap` | Dict-like class for creating a single view of multiple mappings |
| `Counter` | Dict subclass for counting hashable objects |
| `OrderedDict` | Dict subclass that remembers insertion order |
| `defaultdict` | Dict subclass that calls a factory function for missing values |
| `UserDict` | Wrapper around dict objects for easier subclassing |
| `UserList` | Wrapper around list objects for easier subclassing |
| `UserString` | Wrapper around string objects for easier subclassing |

## ChainMap Objects

```python
class collections.ChainMap(*maps)
```

Groups multiple dicts/mappings to create a single, updateable view. Lookups search mappings successively; writes/deletions only operate on the first mapping.

### Key Methods

- **`maps`**: User-updateable list of mappings (ordered first-searched to last-searched)
- **`new_child(m=None, **kwargs)`**: Returns new ChainMap with new map at front
- **`parents`**: Property returning all maps except the first

### Example
```python
baseline = {'music': 'bach', 'art': 'rembrandt'}
adjustments = {'art': 'van gogh', 'opera': 'carmen'}
list(ChainMap(adjustments, baseline))
# ['music', 'art', 'opera']
```

## Counter Objects

```python
class collections.Counter(*args, **kwargs)
```

Dict subclass for counting hashable objects. Elements stored as keys, counts as values.

### Key Methods

- **`elements()`**: Iterator over elements repeated by count
- **`most_common(n=None)`**: List of n most common elements and counts
- **`subtract(iterable/mapping)`**: Subtract counts
- **`total()`**: Sum of all counts (added in 3.10)
- **`update(iterable/mapping)`**: Add counts instead of replacing

### Example
```python
>>> cnt = Counter()
>>> for word in ['red', 'blue', 'red', 'green', 'blue', 'blue']:
...     cnt[word] += 1
>>> cnt
Counter({'blue': 3, 'red': 2, 'green': 1})

>>> Counter('abracadabra').most_common(3)
[('a', 5), ('b', 2), ('r', 2)]
```

### Mathematical Operations
```python
>>> c = Counter(a=3, b=1)
>>> d = Counter(a=1, b=2)
>>> c + d                      # add: c[x] + d[x]
Counter({'a': 4, 'b': 3})
>>> c - d                      # subtract (positive only)
Counter({'a': 2})
>>> c & d                      # intersection: min(c[x], d[x])
Counter({'a': 1, 'b': 1})
>>> c | d                      # union: max(c[x], d[x])
Counter({'a': 3, 'b': 2})
```

## deque Objects

```python
class collections.deque([iterable[, maxlen]])
```

Double-ended queue supporting O(1) appends/pops from either end.

### Key Methods

- **`append(item)`**: Add to right side
- **`appendleft(item)`**: Add to left side
- **`pop()`**: Remove and return from right
- **`popleft()`**: Remove and return from left
- **`extend(iterable)`**: Extend right side
- **`extendleft(iterable)`**: Extend left side (reverses order)
- **`rotate(n=1)`**: Rotate n steps right (negative = left)
- **`reverse()`**: Reverse in-place
- **`clear()`**: Remove all elements
- **`copy()`**: Shallow copy

### Attributes
- **`maxlen`**: Maximum size or None if unbounded

### Common Recipe - Moving Average:
```python
def moving_average(iterable, n=3):
    it = iter(iterable)
    d = deque(itertools.islice(it, n-1))
    d.appendleft(0)
    s = sum(d)
    for elem in it:
        s += elem - d.popleft()
        d.append(elem)
        yield s / n
```

## defaultdict Objects

```python
class collections.defaultdict(default_factory=None, /, **kwargs)
```

Dict subclass that calls a factory function to supply missing values.

### Examples

**Grouping with lists:**
```python
>>> s = [('yellow', 1), ('blue', 2), ('yellow', 3), ('blue', 4), ('red', 1)]
>>> d = defaultdict(list)
>>> for k, v in s:
...     d[k].append(v)
>>> sorted(d.items())
[('blue', [2, 4]), ('red', [1]), ('yellow', [1, 3])]
```

**Counting with int:**
```python
>>> s = 'mississippi'
>>> d = defaultdict(int)
>>> for k in s:
...     d[k] += 1
>>> sorted(d.items())
[('i', 4), ('m', 1), ('p', 2), ('s', 4)]
```

## namedtuple() Factory Function

```python
collections.namedtuple(typename, field_names, *, rename=False, defaults=None, module=None)
```

Returns new tuple subclass with named fields accessible by attribute or index.

### Parameters
- **`typename`**: Name of the new class
- **`field_names`**: Sequence of strings or space/comma-separated string
- **`rename`**: If True, replace invalid fieldnames with positional names
- **`defaults`**: Iterable of default values (applied to rightmost parameters)
- **`module`**: Module name for `__module__` attribute

### Key Methods
- **`_make(iterable)`**: Class method to create instance from iterable
- **`_asdict()`**: Return dict mapping field names to values
- **`_replace(**kwargs)`**: Return new instance with replaced fields

### Key Attributes
- **`_fields`**: Tuple of field names
- **`_field_defaults`**: Dict of field names to default values

### Example
```python
>>> Point = namedtuple('Point', ['x', 'y'])
>>> p = Point(11, y=22)
>>> p[0] + p[1]
33
>>> p._asdict()
{'x': 11, 'y': 22}
>>> p._replace(x=33)
Point(x=33, y=22)

>>> Account = namedtuple('Account', ['type', 'balance'], defaults=[0])
>>> Account._field_defaults
{'balance': 0}
>>> Account('premium')
Account(type='premium', balance=0)
```

## OrderedDict Objects

```python
class collections.OrderedDict(**kwargs)
```

Dict subclass that remembers insertion order.

### Key Methods
- **`popitem(last=True)`**: Remove and return (key, value) pair in LIFO (True) or FIFO (False) order
- **`move_to_end(key, last=True)`**: Move key to end (last=True) or beginning (last=False)

### Differences from dict
- Equality checks are order-sensitive
- Better at frequent reordering operations
- Has efficient `move_to_end()` method

### Recipe: LRU Cache
```python
class TimeBoundedLRU:
    def __init__(self, func, maxsize=128, maxage=30):
        self.cache = OrderedDict()
        self.func = func
        self.maxsize = maxsize
        self.maxage = maxage

    def __call__(self, *args):
        if args in self.cache:
            self.cache.move_to_end(args)
            timestamp, result = self.cache[args]
            if time() - timestamp <= self.maxage:
                return result
        result = self.func(*args)
        self.cache[args] = time(), result
        if len(self.cache) > self.maxsize:
            self.cache.popitem(last=False)
        return result
```

## UserDict, UserList, UserString

Wrapper classes around dict, list, and string objects for easier subclassing.

Each has a `data` attribute containing the underlying real object.
