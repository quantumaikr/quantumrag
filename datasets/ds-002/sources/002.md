# Python `dataclasses` Module Documentation

## Overview

The `dataclasses` module provides a decorator and functions for automatically adding generated special methods like `__init__()` and `__repr__()` to user-defined classes. Originally described in PEP 557.

## @dataclasses.dataclass() Decorator

```python
@dataclasses.dataclass(
    *, init=True, repr=True, eq=True, order=False,
    unsafe_hash=False, frozen=False, match_args=True,
    kw_only=False, slots=False, weakref_slot=False
)
```

**Parameters:**
- **init**: If True (default), generates `__init__()`
- **repr**: If True (default), generates `__repr__()`
- **eq**: If True (default), generates `__eq__()`
- **order**: If True (default False), generates `__lt__()`, `__le__()`, `__gt__()`, `__ge__()`
- **unsafe_hash**: If True, forces `__hash__()` creation
- **frozen**: If True, makes instances immutable
- **match_args**: If True (default), creates `__match_args__` tuple
- **kw_only**: If True, all fields become keyword-only
- **slots**: If True, generates `__slots__` attribute
- **weakref_slot**: If True, adds `__weakref__` slot (requires slots=True)

**Example:**
```python
from dataclasses import dataclass

@dataclass
class InventoryItem:
    name: str
    unit_price: float
    quantity_on_hand: int = 0

    def total_cost(self) -> float:
        return self.unit_price * self.quantity_on_hand
```

## dataclasses.field()

```python
dataclasses.field(
    *, default=MISSING, default_factory=MISSING,
    init=True, repr=True, hash=None, compare=True,
    metadata=None, kw_only=MISSING, doc=None
)
```

**Parameters:**
- **default**: Default value for the field
- **default_factory**: Zero-argument callable for mutable defaults
- **init**: Include in `__init__()`
- **repr**: Include in `__repr__()`
- **hash**: Include in `__hash__()` (uses compare if None)
- **compare**: Include in comparison methods
- **metadata**: Mapping for third-party extensions
- **kw_only**: Marks field as keyword-only
- **doc**: Optional docstring

## InitVar - Init-only Variables

Pseudo-fields passed to `__init__()` and `__post_init__()` but not stored.

```python
@dataclass
class C:
    i: int
    j: int | None = None
    database: InitVar[DatabaseType | None] = None

    def __post_init__(self, database):
        if self.j is None and database is not None:
            self.j = database.lookup('j')

c = C(10, database=my_database)
```

## Key Functions

### fields(class_or_instance)
Returns tuple of Field objects. Does not return ClassVar or InitVar pseudo-fields.

### asdict(obj, *, dict_factory=dict)
Converts dataclass to dict, recursing into nested dataclasses.

```python
@dataclass
class Point:
    x: int
    y: int

p = Point(10, 20)
assert asdict(p) == {'x': 10, 'y': 20}
```

### astuple(obj, *, tuple_factory=tuple)
Converts dataclass to tuple.

### make_dataclass(cls_name, fields, *, bases=(), ...)
Dynamically creates a dataclass.

```python
C = make_dataclass('C',
                   [('x', int), 'y', ('z', int, field(default=5))],
                   namespace={'add_one': lambda self: self.x + 1})
```

### replace(obj, /, **changes)
Creates new instance with specified field values replaced.

### is_dataclass(obj)
Returns True if parameter is a dataclass or instance of one.

## Post-init Processing

`__post_init__()` is called after `__init__()`. Receives InitVar fields as parameters.

```python
@dataclass
class C:
    a: float
    b: float
    c: float = field(init=False)

    def __post_init__(self):
        self.c = self.a + self.b
```

## ClassVar - Class Variables

Fields annotated with `typing.ClassVar` are ignored by dataclasses:

```python
@dataclass
class C:
    x: int
    y: ClassVar[int] = 0  # Not in __init__()
```

## Frozen Instances

```python
@dataclass(frozen=True)
class Point:
    x: float
    y: float

p = Point(1, 2)
p.x = 3  # Raises FrozenInstanceError
```

## Inheritance

Derived class fields override base class fields. Fields combined in MRO order:

```python
@dataclass
class Base:
    x: Any = 15.0
    y: int = 0

@dataclass
class C(Base):
    z: int = 10
    x: int = 15  # Overrides Base.x

# Generated: __init__(self, x: int = 15, y: int = 0, z: int = 10)
```

## KW_ONLY Sentinel

```python
@dataclass
class Point:
    x: float
    _: KW_ONLY
    y: float
    z: float

p = Point(0, y=1.5, z=2.0)
```

## Mutable Default Values

Dataclasses prevent mutable defaults:
```python
@dataclass
class D:
    x: list = []  # ValueError! Use default_factory instead

@dataclass
class D:
    x: list = field(default_factory=list)  # Correct
```

## Descriptor-Typed Fields

```python
class IntConversionDescriptor:
    def __init__(self, *, default):
        self._default = default
    def __set_name__(self, owner, name):
        self._name = "_" + name
    def __get__(self, obj, type):
        if obj is None: return self._default
        return getattr(obj, self._name, self._default)
    def __set__(self, obj, value):
        setattr(obj, self._name, int(value))

@dataclass
class InventoryItem:
    quantity_on_hand: IntConversionDescriptor = IntConversionDescriptor(default=100)

i = InventoryItem()
print(i.quantity_on_hand)   # 100
i.quantity_on_hand = 2.5    # calls __set__ with 2.5
print(i.quantity_on_hand)   # 2
```
