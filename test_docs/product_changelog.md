# QuantumRAG 제품 변경 이력

## v2.5.0 (2024-10-15)

### 새로운 기능
- **HyPE (Hypothetical Prompt Embedding)** 도입
  - 청크당 가상 질문 자동 생성
  - 검색 정확도 15% 향상 (내부 벤치마크 기준)
- Google Gemini 모델 지원 추가
- CSV 파일 파서 추가

### 개선
- BM25 검색 속도 40% 개선 (tantivy 엔진 도입)
- 임베딩 배치 처리 최적화 (처리 속도 2배)
- 한국어 형태소 분석기 Kiwi 통합

### 버그 수정
- PDF 파싱 시 표 데이터 누락 문제 해결
- 대용량 파일(100MB+) 인제스트 시 메모리 초과 문제 수정
- Windows 환경에서 경로 처리 오류 수정

### 호환성 주의
- Python 최소 버전: 3.10 → 3.11 변경
- tantivy 0.22 이상 필요 (이전 버전 비호환)

## v2.4.0 (2024-07-20)

### 새로운 기능
- **Adaptive Query Routing** 도입
  - SIMPLE/MEDIUM/COMPLEX 3단계 자동 분류
  - SIMPLE 쿼리 응답 속도 60% 개선
- PPTX, XLSX 파서 추가
- Cohere Reranker 지원

### 개선
- 스트리밍 응답 안정성 개선
- 인제스트 진행률 표시 기능
- 에러 메시지 한국어 지원

### 버그 수정
- 동시 쿼리 시 race condition 발생 문제 수정
- UTF-8 BOM이 포함된 파일 파싱 실패 문제 해결
- 청크 오버랩 계산 오류 수정

---

## v2.3.0 (2024-04-10)

### 새로운 기능
- **Triple Index Fusion** 도입 (Original + HyPE + BM25)
  - 이전 단일 벡터 검색 대비 정확도 25% 향상
- Anthropic Claude 모델 지원
- HWP/HWPX 파일 파서 추가

### 개선
- 컨텍스트 압축으로 토큰 사용량 30% 절감
- 문서 메타데이터 자동 추출 개선

### 버그 수정
- 긴 문서 청킹 시 문장 중간 잘림 문제 해결
- SQLite WAL 모드 비활성화 상태에서 동시 접근 오류 수정
