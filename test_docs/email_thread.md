## [메일 3] 2024-09-18 | 발신: 강지훈 (AI팀장) → 박서연 (CTO)

**제목: HyPE 성능 중간 보고**

CTO님,

HyPE 알파 테스트 중간 결과입니다:

| 지표 | 기존 (Original Only) | HyPE 추가 | 개선율 |
|------|---------------------|-----------|--------|
| Recall@5 | 0.72 | 0.83 | +15.3% |
| MRR@5 | 0.65 | 0.78 | +20.0% |
| 인제스트 시간 (1000문서) | 45분 | 78분 | -73.3% |
| 스토리지 (1000문서) | 1.2GB | 2.1GB | +75.0% |

청크당 생성하는 가상 질문 수를 5개에서 3개로 줄이면 인제스트 시간을 약 40% 단축할 수 있으나, Recall@5가 0.79로 약간 하락합니다.

임베딩 배치 처리 최적화도 진행 중이며, 처리 속도를 2배 향상시킬 수 있을 것으로 예상합니다.
