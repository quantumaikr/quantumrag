## 4. SWOT 분석 (퀀텀소프트)

### Strengths (강점)
- Triple Index Fusion 기술 (검색 정확도 업계 최고)
- 한국어 특화 처리 (형태소 분석, HWP 지원)
- 높은 고객 만족도 (NPS 72)
- 낮은 이탈률 (5%)

### Weaknesses (약점)
- 시장 점유율 4위 (11%)
- 자체 LLM 미보유 (외부 API 의존)
- 온프레미스 배포 미지원 (2025년 지원 예정)
- 직원 수 87명으로 경쟁사 대비 소규모

### Opportunities (기회)
- RAG 시장 급성장 (CAGR 52%)
- 일본 시장 진출 (NTT 데이터 파트너십)
- 금융/공공 섹터 수요 증가
- Series C를 통한 스케일업

### Threats (위협)
- 퀀텀아이의 자체 LLM + RAG 통합 전략
- 글로벌 업체 (Pinecone, Weaviate) 국내 진출
- OpenAI API 비용 상승 리스크
- 대기업(네이버, 카카오) 자체 RAG 개발
