# QuantumRAG 고객 사례 보고서

## 사례 1: 삼성전자 반도체 사업부

### 도입 배경
- 반도체 공정 문서 약 50,000건 (PDF, 한글, 엑셀 혼재)
- 기존 검색 시스템: 키워드 기반, 검색 만족도 35%
- 엔지니어 1인당 주 평균 8시간을 문서 검색에 소비

### 도입 제품
- QuantumRAG Enterprise (월 800만원)
- 온프레미스 배포 (자체 GPU 서버)
- 커스텀 한국어 임베딩 모델 적용

### 성과 (도입 6개월 후)
| 지표 | 도입 전 | 도입 후 | 개선율 |
|------|--------|---------|--------|
| 검색 만족도 | 35% | 82% | +134% |
| 문서 검색 시간 | 주 8시간 | 주 2.5시간 | -69% |
| 정확 답변율 | 28% | 76% | +171% |
| 월 쿼리 수 | 12,000 | 45,000 | +275% |

### 피드백
- "PDF 내 표 데이터 검색이 획기적으로 개선됨" (김 과장, 공정개발팀)
- "한국어-영어 혼합 문서도 정확하게 검색됨" (이 차장, 품질관리팀)
- **개선 요청**: 스캔된 PDF(이미지) 지원, 도면 내 텍스트 검색

## 사례 2: KB국민은행

### 도입 배경
- 고객 상담 FAQ 2,300건 + 내부 규정 문서 1,200건
- 상담원 평균 응답 시간: 4.2분
- 오답률: 15% (신입 상담원 기준 28%)

### 도입 제품
- QuantumRAG Pro (월 300만원) + QuantumChat (월 300만원)
- 클라우드 배포 (AWS ap-northeast-2)
- Slack, Teams 연동

### 성과 (도입 3개월 후)
| 지표 | 도입 전 | 도입 후 | 개선율 |
|------|--------|---------|--------|
| 평균 응답 시간 | 4.2분 | 1.8분 | -57% |
| 오답률 | 15% | 4.2% | -72% |
| 상담원 만족도 (NPS) | 31 | 67 | +116% |
| 일 처리 건수 | 180건 | 290건 | +61% |

### 피드백
- "신입 상담원 교육 기간이 2주에서 3일로 단축됨" (박 팀장, 고객센터)
- **개선 요청**: 응답 속도 1.5초 이하 목표 (현재 평균 2.8초), 실시간 응답 스트리밍

## 사례 3: 네이버 클라우드

### 도입 배경
- 기술 문서(API 레퍼런스, 가이드) 8,500건
- 개발자 포럼 질문 응답 자동화 필요
- 경쟁사(Upstage) 솔루션에서 전환

### 도입 제품
- QuantumRAG Enterprise (월 1,200만원)
- 자체 클라우드 배포
- 커스텀 기술 도메인 파인튜닝

### 성과 (도입 9개월 후)
| 지표 | 도입 전 (Upstage) | 도입 후 (QuantumRAG) | 개선율 |
|------|-------|------|--------|
| 검색 정확도 (NDCG@5) | 0.68 | 0.81 | +19% |
| 평균 응답 속도 | 3.1초 | 2.3초 | -26% |
| 사용자 만족도 | 62% | 84% | +35% |
| 월간 활성 사용자 | 3,200 | 5,100 | +59% |

### 피드백
- "Triple Index Fusion이 기술 문서 검색에서 압도적 성능" (최 리드, 플랫폼팀)
- "Upstage 대비 한국어 처리 능력이 확실히 우수" (정 매니저, DevRel팀)
- **개선 요청**: 코드 스니펫 검색 특화 기능, GraphRAG 지원

## 사례 4: 현대자동차 연구소

### 도입 배경
- 차량 기술 규격서 및 시험 성적서 약 30,000건
- 다국어 문서 (한국어 60%, 영어 30%, 독일어 10%)
- 기존 사내 검색 시스템 노후화 (10년 이상 운영)

### 도입 제품
- QuantumRAG Pro (월 300만원) — PoC 단계
- QuantumChat (월 300만원) — PoC 단계
- 기간: 2024년 11월 ~ 2025년 2월 (3개월 PoC)

### PoC 중간 결과 (1개월 차)
| 지표 | 기존 시스템 | QuantumRAG | 개선율 |
|------|-----------|------------|--------|
| 한국어 문서 정확도 | 45% | 78% | +73% |
| 영어 문서 정확도 | 52% | 81% | +56% |
| 다국어 혼합 검색 | 불가 | 72% | N/A |
| 평균 응답 속도 | 5.8초 | 2.1초 | -64% |

### 피드백
- "다국어 혼합 검색이 가능하다는 것 자체가 혁신" (강 박사, 연구기획팀)
- **개선 요청**: 독일어 형태소 분석 지원, 도면 내 부품 번호 검색
- **리스크**: PoC 실패 시 Upstage로 재전환 가능성 있음

## 사례 5: 법무법인 김앤장

### 도입 배경
- 판례 DB 120,000건 + 법률 자문 의견서 25,000건
- 변호사 1인당 주 12시간을 판례 검색에 투자
- 보안 요구: 모든 데이터 국내 저장, 외부 전송 불가

### 도입 제품
- QuantumRAG Enterprise (월 2,000만원)
- 완전 온프레미스 배포 (Ollama 로컬 모델 사용)
- 법률 도메인 특화 파인튜닝

### 성과 (도입 4개월 후)
| 지표 | 도입 전 | 도입 후 | 개선율 |
|------|--------|---------|--------|
| 판례 검색 정확도 | 40% | 88% | +120% |
| 검색 시간 | 주 12시간 | 주 3시간 | -75% |
| 관련 판례 누락률 | 35% | 8% | -77% |

### 피드백
- "형사 판례와 민사 판례 간 교차 검색이 가능해져 업무 품질이 향상됨"
- **최대 고객 (월 매출 2,000만원)** — 퀀텀소프트 전체 매출의 약 16%
- **개선 요청**: 법률 인용 형식 자동 생성, 판례 간 관계 시각화
