## Q1 2025 (1월~3월)

### QuantumRAG v3.0 — "Atlas" 릴리스
- **[P0] OCR 통합**: 스캔된 PDF 자동 텍스트 추출 (Tesseract + 자체 후처리)

- **[P1] 중복 문서 감지**: 해시 + 임베딩 유사도 기반
- **[P2] 멀티모달 검색**: 이미지 캡션 기반 검색 (삼성전자 요청)

**의존성**: OCR 통합은 AI팀 2명 + 인프라팀 1명 필요 (3개월)

### QuantumChat v2.0
- **[P0] 멀티 에이전트 지원**: 부서별 전문 에이전트
- **[P1] 음성 입력/출력**: STT/TTS 연동
