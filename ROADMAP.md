# QuantumRAG Roadmap

## v0.2 - Production Hardening (Q2 2026)

- [ ] Broad exception narrowing (`except Exception` -> specific types)
- [ ] End-to-end integration test suite
- [ ] Docker Compose one-click deployment
- [ ] Streaming query API (SSE)
- [ ] Multi-tenant ACL filtering
- [ ] Plugin marketplace (community connectors)
- [ ] Performance benchmarks (vs LangChain, LlamaIndex)

## v0.3 - Enterprise Features (Q3 2026)

- [ ] Role-based access control (RBAC)
- [ ] Audit logging for compliance
- [ ] Distributed vector store support (Qdrant, Milvus, Pinecone)
- [ ] Multi-language support beyond Korean (Japanese, Chinese)
- [ ] Document versioning and change tracking
- [ ] Webhook notifications for ingest events
- [ ] Admin dashboard (web UI)

## v1.0 - Stable Release (Q4 2026)

- [ ] Stable public API (semantic versioning guarantee)
- [ ] Comprehensive documentation site
- [ ] Official Helm chart for Kubernetes
- [ ] SOC 2 compliance preparation
- [ ] LTS support commitment
- [ ] Migration guides from LangChain / LlamaIndex

---
